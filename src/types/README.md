# Type Definitions (`/src/types`)

## 🎯 Purpose
This directory houses global TypeScript interfaces and type definitions that don't logically belong to a single, specific module.

## 🏛️ Domain Boundaries
* **Owns:** Abstract types, globally shared interfaces, and utility types.
* **Does NOT Own:** Zod runtime schemas (see `/src/schemas`).

## 🔑 Key Entry Files
* Shared type definition files.

## 🔄 Dependencies
* **Upstream (Depends on):** None.
* **Downstream (Depended on by):** Entire codebase.
